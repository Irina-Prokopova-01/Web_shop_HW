# WebShop

## Описание:
Проект он-лайн магазина, реализуемый в рамках выполнения домашних заданий. Реализован функциональный вид следующих страниц:

* главная;
* каталог;
* категория 1;
* контакты. 
* Реализован скрипт сервера, обрабатывающий GET и POST запросы.
